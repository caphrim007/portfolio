About

resolver is an application developed to do real-time name resolution of netflow generated by site networking equipment.

Features

The following are key features of resolver
  • Ability to resolve, in real-time, 24+ gigabytes of netflow a day
  • Name resolution speed on the order of 500,000+ addresses a minute
  • Low memory usage; on the order of 60 megabytes
  • Generation of logs for charting and metrics
  • Simple scalability via multiple instances of node

Technologies Used

Overall, the application is rather simple in design; consisting primarily of server-side javascript
  • node.js
  • splunk
  • dnsmasq
  • Linux
  • OSU flow-tools
