About

Solera DeepSee App for Splunk is an application developed to integrate Solera's DeepSee appliance functionality in the Splunk UI.

Features

The following are key features of Solera DeepSee App for Splunk
  • Connects events found in Splunk to the full packet capture and reconstruction of the event in Solera DS Appliances

Technologies Used

This is an app built on top of the Splunk framework. It includes custom event renderers, javascript and manipulation of the UI that reaches far beyond what a normal Splunk app does.
  • Splunk
  • Python
  • Javascript
  • HTML
