About

nessquik is an application developed to act as a frontend for the Nessus Vulnerability Scanner sold by Tenable.

Features

The following are key features of nessquik
  • Ability to scna an arbitrary number of hosts with an arbitrary number of policies
  • No Flash requirement
  • Support for controlling all versions of Nessus including the 3.x, 4.x, and 5.x lines simultaneously
  • Custom reporting tools that extend beyond what the default Nessus client provides
  • XML-RPC API able to control URL creation and facilitate creation of 3rd party inject tools
  • Advanced authentication support and permissions system allows fine-grained access to savory. Authentication supports integration with your existing authentication systems

Technologies Used

The application is modular in design and extendable, allowing for customization specific to your site.
