About

Solera Greasemonkey Framework is a set of Greasemonkey scripts that integrate Solera's DeepSee appliance functionality in any webpage of your choosing.

Features

The following are key features of Solera Greasemonkey Framework
  • Connects IP addresses and ports found in webpages to the full packet capture and reconstruction of those tuple items in Solera DS Appliances

Technologies Used

This is an app built on top of the Greasemonkey Firefox extension. It primarily includes standard web technologies, but required some development knowledge that is specific to the Greasemonkey realm due to operating in a sandboxed environment.
