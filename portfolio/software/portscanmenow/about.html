About

portscan-me-now is a web frontend for the Nmap port scanner. It focuses on simplicity.

Features

The following are key features of portscan-me-now
  • Point and click port scanning of devices
  • Safe, in-that only the machine contacting portscan-me-now is scanned
  • Reports are held for 24 hours
  • Options for an internal, and an external (such as DMZ) scanner

Technologies Used

Overall, the application is rather simple in design; consisting primarily of server-side javascript
  • Linux
  • cherrpy
  • Python
  • PHP
  • API
  • REST
  • JSON
