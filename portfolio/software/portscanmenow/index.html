What it is
portscanmenow is a web frontend for the Nmap port scanner. It focuses on simplicity. Have a look
What I did
  • Complete design and implementation of system, encompassing the entire software development lifecycle
  • Integration with backend NMAP command line interface
  • Custom reporting interface for different groups onsite
  • Advanced multi-tier authentication system supporting LDAP and X.509 certificates
  • Point-and-shoot application to make port scans of your own systems as easy as possible
